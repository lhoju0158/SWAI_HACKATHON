2024년 6월 27일에 수행한 SWAI_HACKATHON에 관한 코드이다.

주제: Social Impact: 미래 사회문제 예측 및 해결방안 제시

치안을 위협받는 상황에서, 신고 이외에 불안감 해소 및 범죄예방을 할 수 있는 수단 제공
