2024년 6월 27일에 수행한 SWAI_HACKATHON에 관한 코드이다.

주제: Social Impact: 미래 사회문제 예측 및 해결방안 제시

치안을 위협받는 상황에서, 신고 이외에 불안감 해소 및 범죄예방을 할 수 있는 수단 제공 행정안전부의 생활안전지도 데이터, 사용자 위치로그에 기반해 치안 불안감 해소와 범죄예방목적의 위험 타이머 리포트 어플 개발
